# Сайт об Артёмовске - История и Культура

Многостраничный веб-сайт, посвящённый истории и культуре города Артёмовск.

## Описание проекта

Этот проект создан в рамках лабораторной работы №1 по курсу "Web-программирование". Сайт демонстрирует применение современных технологий HTML5 и CSS3 для создания информационного веб-ресурса о городе.

## Использованные технологии

- **HTML5** - семантическая разметка, современные теги
- **CSS3** - стилизация, адаптивный дизайн, градиенты, анимации
- **JavaScript** - валидация формы обратной связи
- **Git/GitHub** - система контроля версий

## Структура сайта

Сайт состоит из 6 страниц:

1. **index.html** - Главная страница с общей информацией о городе и фотогалереей
2. **about.html** - Страница о разработчике с биографией и профессиональными навыками
3. **history.html** - История города с таблицей из 10+ исторических событий
4. **infrastructure.html** - Инфраструктура города (транспорт, коммунальные услуги)
5. **culture.html** - Культура и искусство (театры, музеи, фестивали)
6. **contact.html** - Форма обратной связи с валидацией

## Технические требования

Проект соответствует всем требованиям лабораторной работы:

- Использовано **55+ различных HTML-тегов**
- Применено **минимум 15 тегов HTML5** (nav, header, footer, section, article и др.)
- Использовано **3 способа позиционирования**: static, relative, sticky
- Применён **минимум 1 способ обтекания**: float

## Как запустить сайт локально

1. Скачайте или клонируйте репозиторий:
```bash
git clone https://github.com/Destizer/artemovsk-site.git
```

2. Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Safari, Edge)

3. Навигация по сайту осуществляется через меню в верхней части страницы
